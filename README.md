# 🧬 BioAI Insight Engine

An end-to-end AI-powered cancer risk prediction system integrating Machine Learning, Deep Learning, Explainable AI, and an interactive web dashboard.

🌍 **Live App:** https://bioai-insightengine-gdhzbdpdthtjzw8aemj9d7.streamlit.app/

---

## 🚀 Project Overview

BioAI Insight Engine is a biotechnology-focused AI system that:

- Predicts cancer risk using biological feature inputs
- Uses Machine Learning (Random Forest, Logistic Regression, SVM)
- Implements Deep Learning using PyTorch
- Provides Explainable AI via Feature Importance
- Deploys an interactive dashboard using Streamlit
- Supports research paper semantic search (RAG system)

This project demonstrates a full AI pipeline from data preprocessing to cloud deployment.

## 🖥️ Run Locally

Clone repository:

```bash
git clone https://github.com/rejoy2004-rgb/BioAI-Insight-Engine.git
cd BioAI-Insight-Engine
```

Create environment:

```bash
conda create -n bioai python=3.10
conda activate bioai
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run app:

```bash
streamlit run app.py
```
